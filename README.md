# CBrowser

**CBrowser** is a simple web browser built in **C++** using **Qt**.  
This project is designed as an easy-to-understand desktop browser implementation — great for learning how browsers work and for experimenting with Qt’s web capabilities. :contentReference[oaicite:1]{index=1}

---

##  Features

- Basic graphical browser interface  
- Navigation controls (back, forward, reload)  
- Simple bookmarks support - WIP
- Built with Qt and WebEngine components

_(More advanced features may be added over time)_
